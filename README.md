# nand2tetris_mini_platformer_project
I created a mini platformer using nand2tetris tool for my university project.

You control a tiny cube as you try to beat the game. The game has 3 mini levels filled with spikes and turrets.
It also tracks your "time", it actually tracks frames but they are used as score.
You control cube with A(move left), D(move right) and W(jump).


## How to Run : 
1. Download Nand2Tetris tools
2. Open `VMEmulator.bat`
3. Load the project folder
4. Set:
   - Animate → No animation
   - Speed → Fast
   - View → Screen
5. Press Run
